River

River is a dynamic tiling Wayland compositor with flexible runtime configuration.

Keybindings

Here's a list of some important keybinds you need to know for operating river easily. If you want to view, modify or add new keybinds, Edit ~/.config/river/init file.

  • W : Window
  • C : Control
  • S : Shift
  • A : Alt

Applications

Keybind Action
W + Return Open terminal (alacritty)
W + S + Return Open floating terminal
W + A + Return Open fullscreen terminal
W + S + f Open file manager (thunar)
W + S + e Open text editor (geany)
W + S + w Open web browser (firefox)

Menu and Applets

Keybind Action
W + d Open application launcher
W + r Open command runner
W + n Open network manager
W + b Open bluetooth applet
W + m Open media player
W + x Open powermenu
W + s Open screenshot menu

Window Management

Keybind Action
W + c/q Close application
W + Left/Right Focus next/previous view in the layout stack
W + S + Left/Right Swap focused view with the next/previous view in the layout stack
W + j/k Focus next/previous output
W + S + j/k Send the focused view to next/previous output
W + e Bump the focused view to the top of the layout stack
W + h/l Decrease/increase the main_factor value of rivertile by 0.02
W + S + h/l Increment/decrement the main_count value of rivertile
C + A + Left/Down/Up/Right Change layout orientation accordingly
W + A + Left/Down/Up/Right Move views (floating) accordingly
W + C + Left/Down/Up/Right Resize views accordingly
W + A + C + h/j/k/l Snap views to screen edges
W + f Toggle fullscreen mode
W + Space Toggle between floating and tiled mode

Misc Keys

Keybind Action
W + p Color Picker
C + A + l Lockscreen
C + A + Space Dismiss notifications
C + A + Delete Quit River session

Hardware Keys

Keybind Action
Print Take screenshot of desktop
A + Print Take screenshot of desktop in 5 seconds
S + Print Take screenshot of desktop in 10 seconds
W + Print Take screenshot of selected area
XF86AudioRaiseVolume Increase Volume
XF86AudioLowerVolume Decrease Volume
XF86AudioMute Toggle Mute Speakers
XF86AudioMicMute Toggle Mute Microphone
XF86Audio + Next/Prev/Play/Stop Media Controls (MPD)
XF86MonBrightnessUp Increase Screen Brightness
XF86MonBrightnessDown Decrease Screen Brightness

Themes

pywal support has been added to river. To change themes, Left Click on the theme module on waybar. To restore the default theme, Right Click on the same module on waybar.
