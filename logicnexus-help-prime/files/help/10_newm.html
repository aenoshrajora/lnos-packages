NEWM

NEWM is a Wayland compositor written with laptops and touchpads in mind. The idea is, instead of placing windows inside the small viewport (that is, the monitor) to arrange them along an arbitrarily large two-dimensional wall (generally without windows overlapping) and focus the compositors job on moving around along this wall efficiently and providing ways to the user to rearrange the wall such that they find the overall layout intuitive.

Keybindings

Here's a list of some important keybinds you need to know for operating newm easily. If you want to view, modify or add new keybinds, Edit ~/.config/newm/config.py file.

  • W : Window
  • C : Control
  • S : Shift
  • A : Alt

Applications

Keybind Action
W + Return Open terminal (foot)
W + S + Return Open floating terminal
W + A + Return Open terminal with selected area
W + f Open file manager (thunar)
W + e Open text editor (geany)
W + w Open web browser (firefox)

Menu and Applets

Keybind Action
W + d Open application launcher
W + r Open command runner
W + n Open network manager
W + b Open bluetooth applet
W + m Open media player
W + x Open powermenu
W + s Open screenshot menu

Window Management

Keybind Action
Window/Super Show Overview
W + c/q Close application
W + Left/Right/Up/Down Change focus of the window directionally
W + S + Left/Right/Up/Down Move windows directionally
W + C + Left/Right/Up/Down Resize windows directionally
W + a Move in stack
W + Space Toggle fullscreen
W + S + Space Toggle floating
W + Minus Scale down
W + Equal Scale Up

Gestures

Keybind Action
W + MOVE_1 Move windows
W + SWIPE_2 Resize windows
SWIPE_3 Swipe within the 2D workspace
SWIPE_4 Scale up and down

Misc Keys

Keybind Action
W + p Color Picker
C + A + l Lockscreen
W + S + c Reload the configuration file
W + S + q Quit NEWM session

Hardware Keys

Keybind Action
Print Take screenshot of desktop
A + Print Take screenshot of desktop in 5 seconds
S + Print Take screenshot of desktop in 10 seconds
W + Print Take screenshot of selected area
XF86AudioRaiseVolume Increase Volume
XF86AudioLowerVolume Decrease Volume
XF86AudioMute Toggle Mute Speakers
XF86AudioMicMute Toggle Mute Microphone
XF86MonBrightnessUp Increase Screen Brightness
